# Проект: Основы ООП — Интернет-магазин

Проект реализует базовую логику управления товарами и категориями. Система построена на принципах объектно-ориентированного программирования (инкапсуляция, работа с атрибутами классов и экземпляров) и включает автоматизированную загрузку данных.

---

## 🛠 Основной функционал

### 🔹 Классы и объекты
*   **Product**: Описывает товар. 
    *   Инкапсуляция цены через приватный атрибут.
    *   Сеттер цены с проверкой (запрет ≤ 0) и подтверждением снижения стоимости через консоль.
    *   Класс-метод `new_product` для "умного" создания объекта или обновления существующего в списке (складывание остатков и выбор максимальной цены).
*   **Category**: Группирует товары. 
    *   Приватный список товаров, защищенный от прямого изменения.
    *   Геттер `products`, возвращающий красиво форматированную строку со списком всех товаров.

### 🔹 Автоматическая статистика
Класс `Category` отслеживает состояние склада в атрибутах класса:
*   `category_count`: Общее количество созданных категорий.
*   `product_count`: Общее количество уникальных товаров во всех категориях.

### 🔹 Инструменты и загрузка данных
*   **read_json**: Функция в `utils.py`, которая читает файл и преобразует данные напрямую в объекты классов `Category` и `Product`.
*   **Типизация**: Весь код аннотирован типами и проходит проверку `mypy`.

---

## 📂 Структура проекта
- `src/product.py` — Логика работы с товарами, геттеры и сеттеры.
- `src/category.py` — Управление категориями и учет статистики.
- `src/utils.py` — Загрузка данных из внешних JSON-файлов.
- `tests/` — Полный набор модульных тестов.

---

## 🚀 Запуск и тестирование

### Установка окружения
Рекомендуется использовать Python 3.10+ и виртуальное окружение.
```bash
pip install pytest pytest-cov mypy flake8
```

### Запуск проекта
Для корректной работы импортов запуск производится из корня проекта:
```bash
python -m src.utils
```

### Тестирование и статический анализ
Проект следует стандартам качества кода:
*   **Тесты**: `pytest --cov=src` (проверка логики и покрытия).
*   **Типы**: `mypy src` (проверка статической типизации).
*   **Стиль**: `flake8 src` (проверка соответствия PEP 8).
