# Проект: Основы ООП — Интернет-магазин

Проект реализует базовую логику управления товарами и категориями. Включает в себя автоматический учет статистики, загрузку данных из внешних файлов и полное покрытие тестами.

---

## 🛠 Основной функционал

### 🔹 Классы и объекты
*   **Product**: Описывает конкретный товар. Хранит название, описание, цену и остаток на складе.
*   **Category**: Объединяет товары в группы. Хранит название, описание и список объектов `Product`.

### 🔹 Автоматическая статистика
Класс `Category` самостоятельно ведет учет данных в атрибутах класса:
*   `category_count`: общее количество созданных категорий.
*   `product_count`: общее количество уникальных товаров во всех категориях.

### 🔹 Загрузка из JSON
Реализована функция `read_json`, которая:
1. Читает файл формата `.json`.
2. Автоматически создает экземпляры классов `Product` и `Category`.
3. Формирует готовую структуру данных для работы.

---

## 📂 Структура файлов
- `src/product.py` — описание класса товара.
- `src/category.py` — описание класса категории и счетчиков.
- `src/utils.py` — логика чтения файлов и инициализации объектов.
- `tests/` — набор тестов для проверки каждого модуля.

---

## 🧪 Тестирование и покрытие
В проекте используется **Pytest** и **Pytest-cov**. Написаны тесты для:
- Проверки корректности инициализации атрибутов.
- Контроля работы глобальных счетчиков категорий и товаров.
- Валидации процесса загрузки данных из JSON.
